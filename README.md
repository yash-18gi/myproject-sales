# Sales Data Analysis using Python

## 📖 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on retail sales data to uncover meaningful business insights. The analysis focuses on customer demographics, purchasing behavior, product performance, payment methods, and revenue trends.

The project demonstrates the use of Python for data cleaning, transformation, analysis, and visualization to support data-driven business decisions.

---

## 🎯 Objectives

- Clean and preprocess raw sales data.
- Analyze customer and sales information.
- Identify purchasing patterns and revenue trends.
- Visualize key business metrics.
- Generate actionable insights from the dataset.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

The project uses two datasets:

- **Customer Data**
- **Sales Data**

These datasets are combined using a common customer identifier to perform comprehensive analysis.

---

## 🔍 Data Preprocessing

The following preprocessing steps were performed:

- Checked data types
- Handled missing values
- Removed duplicate records
- Converted date columns to datetime format
- Created new calculated columns where necessary
- Validated numerical values
- Generated descriptive statistics

---

## 📊 Exploratory Data Analysis

The analysis includes:

- Customer Gender Distribution
- Customer Age Distribution
- Product Category Analysis
- Payment Method Analysis
- Revenue by Product Category
- Revenue by Payment Method
- Quantity Sold Analysis
- Correlation Analysis
- Identification of Outliers
- Summary Statistics

---

## 📈 Visualizations

The project includes visualizations created using **Matplotlib**, such as:

- Bar Charts
- Pie Charts
- Box Plots
